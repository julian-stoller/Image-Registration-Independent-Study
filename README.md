# Image-Registration-Independent-Study
A collection of basic but increasingly involved image registration problems with provided solutions and explanations. Intended to document my independent study of the subject and be a resource for future students.

### Course Structure:
- **Black Square On A White Background**
  - Translational
  - Rotational
  - Translational + Rotational
  - Scale
  - Translational + Rotational + Scale
  - Different Colors
  - What do you do for non-uniform or noisy backgrounds?
